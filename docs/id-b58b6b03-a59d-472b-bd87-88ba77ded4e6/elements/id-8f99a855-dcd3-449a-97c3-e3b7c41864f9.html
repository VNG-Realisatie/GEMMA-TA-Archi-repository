Toepasbare regelscomponent

Naast de juridische regels stelt de gemeente ook toepasbare regels op. Bijvoorkeur is dit een iteratief proces waarbij juridische en toepasbare regels direct naast elkaar worden gemaakt. Hierbij zorgt de gemeente dat juridische en toepasbare regels goed op elkaar aansluiten en aan blijven sluiten. Leveranciers van software voor juridische en toepasbare regels bieden verschillende opleidingen om beide type regels met elkaar in sync te houden.
