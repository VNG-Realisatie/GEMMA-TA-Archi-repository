Generiek zaakafhandelcomponent ()
Component voor het afhandelen van zaken van alle typen.
ToelichtingEen generieke component die medewerkers ondersteunt bij het afhandelen van verschillende soorten zaken. Belangrijk is dat de Generieke Zaakafhandelcomponent geschikt is om zaken van alle typen af te handelen. Dus niet beperkt tot zaken van één domein.
