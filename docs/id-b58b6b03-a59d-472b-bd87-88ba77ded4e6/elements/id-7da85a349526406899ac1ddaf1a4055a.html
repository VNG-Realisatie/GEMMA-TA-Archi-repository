Beheerportaal BRK-PB ()
Beheerportaal voor het registreren en bijhouden van publiekrechtelijke beperkingen.
BBNBBN 1 en BBN2 integriteitsmaatregelen
BIV score BBN121
Beschikbaarheid1
Beschikbaarheid (belangrijkste reden)1 : interne negatieve publiciteit (imagoschade).
ToelichtingHet registreren en bijhouden van een publiekrechtelijke beperking is mogelijk via het BRK-PB beheerportaal. Het portaal is via Mijn Kadaster benaderbaar. Voor het inloggen op Mijn Kadaster is eHerkenning op minimaal niveau EH3 vereist. Wanneer ook andere organisaties, bijvoorbeeld een Omgevingsdienst de werkzaamheden voor u uitvoeren dan kunnen zij via een ketenmachtiging toegang krijgen tot uw portaal.
